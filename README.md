# TenseLine

TenseLine is an interactive, local-first grammar timeline maker for teachers. Create colour-coded moments and continuous actions, arrange them on a movable past–present–future timeline, and connect each event to selected words in an example sentence or paragraph.

## Features

- Add events by clicking directly on the timeline.
- Drag events and the “Now” marker freely.
- Display an event as a single moment or an adjustable continuous action.
- Automatically stack nearby event labels to keep them readable.
- Link event colours to words in a sentence or paragraph.
- Increase or decrease text size for classroom projection.
- Use presentation mode for a clean teaching view.
- Save work automatically in the current browser.

## Publish with GitHub Pages

1. Create a new public GitHub repository, for example `tenseline`.
2. Upload all files from this package to the root of the repository.
3. Open the repository’s **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder, then click **Save**.
6. GitHub will provide a link similar to `https://YOUR-USERNAME.github.io/tenseline/`.

No build command or server is required. The site consists of plain HTML, CSS, and JavaScript.

## Local use

Open `index.html` in a modern browser. Timeline data is stored locally on that device.
